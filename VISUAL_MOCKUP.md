## Animations & Effects

### Glow Effect (Active Elements)
- 8-step radial gradient
- 30% base opacity
- Green color (77, 255, 102)
- Fades outward
- Used on: Active sidebar buttons, checked checkboxes, enabled toggles

### Hover Effects
- Border color brightens (200ms transition)
- Background lightens slightly (150ms transition)
- Tooltip appears after 500ms

### Gradients
- Sliders: Light green â†’ Bright green (left to right)
- Buttons: Subtle vertical gradient on hover
- Background: Flat (no gradient)

## Window Sizes

- **Default**: 1280 x 800 pixels
- **Minimum**: 800 x 600 pixels
- **Recommended**: 1100 x 750 pixels
- **Sidebar**: 80px wide (fixed)
- **Content**: Remaining width minus padding

---

## Typography

- **Font**: System default (can use Inter, Roboto, or other modern sans-serif)
- **Size**: 14px body, 16px headers
- **Weight**: Regular for body, Medium for headers
- **Color**: Off-white (242, 242, 242) for primary text
- **Line Height**: 1.5x for readability

---

## Spacing & Padding

- **Window Padding**: 15px all sides
- **Item Spacing**: 12px vertical, 8px horizontal
- **Frame Padding**: 8px horizontal, 6px vertical
- **Indent**: 25px for nested items
- **Section Spacing**: 20px between major sections
